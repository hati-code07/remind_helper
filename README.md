# REMIND HELPER 🤖

Telegram бот для управления учебными задачами и расписанием.

## Возможности

- ✅ Создание задач с дедлайнами
- 📅 Выбор даты через календарь
- ⏰ Напоминания за день, час и в момент дедлайна
- 📋 Расписание пар
- 👑 Админ-панель со статистикой

## Технологии

- Python 3.11
- Aiogram 3.x
- SQLAlchemy 2.0 / SQLite
- APScheduler
- Flask

## Установка

```bash
git clone https://github.com/hati-code07/remind_helper
cd remind_helper
pip install -r requirements.txt
python main.py
